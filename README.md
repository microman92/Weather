# Weather
<img alt="NPM Version" src="https://img.shields.io/npm/v/npm?color=%2330A339"> <img alt="Static Badge" src="https://img.shields.io/badge/node-v18.17.1-%2330A339"> <img alt="Static Badge" src="https://img.shields.io/badge/%40vue%2FCli-5.0.8-%2330A339"> <img alt="Static Badge" src="https://img.shields.io/badge/vue-3-%2330A339">

<p> 
Проект веб-приложения разработан для отображения текущей погоды и времени по заданному пользователем городу. Приложение предоставляет возможность просмотра прогноза погоды на неделю. <br /> Для получения данных о погоде используется API от openweathermap.org, которое обеспечивает достоверную и актуальную информацию о погодных условиях. <br /><br />

Для организации запросов к API и хранения данных в приложении применяется Vuex, <br />  централизованное хранилище состояния данных в приложении Vue.js.  <br /> Сборкой проекта управляет инструмент Vue CLI, обеспечивая удобную разработку и сборку приложения.<br /><br />

Для выполнения сетевых запросов и получения данных о погоде используется встроенный API Fetch, который предоставляет простой и удобный интерфейс для работы с HTTP-запросами в JavaScript.<br />

Такое веб-приложение позволяет пользователям быстро получать информацию о текущей погоде и прогнозе на неделю для выбранного города, обеспечивая удобство и функциональность при использовании. <br />
Приложение адаптировано для корректного отображения на различных устройствах. <br />
В приложении также присутствует 2 темы, светлая и тёмная
</p>
